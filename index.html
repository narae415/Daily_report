북마크릿
Jira가 열린 브라우저에서 북마크릿 클릭 한 번으로 금주 업무 내역이 지라 집계 탭에 자동 반영됩니다.
등록 방법
지라 집계 탭의 ? 버튼에서 북마크릿 등록 가이드를 확인할 수 있습니다.
Jira 업무 활동 내역 추출기

Jira API를 활용해 금주 내 본인이 담당·보고·생성한 티켓 활동 내역을 자동 수집합니다.
사내망 또는 VPN이 연결된 환경, Jira가 로그인된 브라우저에서만 작동합니다.

빠른 사용법
  1. 하단 등록 방법에 따라 북마크릿을 브라우저에 한 번만 등록합니다.
  2. Jira가 열린 상태에서 등록한 북마크릿을 클릭합니다.
  3. 이 페이지가 자동으로 열리며 [지라 집계] 탭에 내역이 바로 반영됩니다.
    메모장 등 다른 곳에 붙여넣기(Ctrl+V)해서 조회하는 것도 가능합니다.
수집 기준
수집 대상금주(월~일) 내 담당/보고/생성한 모든 티켓
분류 기준상태값에 따라 완료 / 업데이트 / 미확인 자동 구분
특수 처리WQA 티켓 설명 추출 · 중복 HELP 티켓 필터링
북마크릿 최초 등록 방법
  1. 아래 [코드 복사] 버튼을 클릭합니다.
  2. 브라우저 북마크 바 우클릭 → [페이지 추가] 선택
  3. 이름(예: Jira 추출) 입력 후 URL 칸에 복사한 코드를 붙여넣기(Ctrl+V)하여 저장합니다.
